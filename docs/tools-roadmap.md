# 工具开发路线图

## 当前已实现 ✅

1. **`list_files`** - 列出目录下的文件和文件夹

## 优先级 1：文件操作（最常用）🔥

### 2. `read_file` - 读取文件内容
- **用途**：读取文本文件内容，让 AI 分析代码、配置等
- **参数**：`path` (文件路径), `encoding` (可选，默认 utf-8), `max_size` (可选，限制读取大小)
- **返回**：文件内容字符串

### 3. `search_in_files` - 在文件中搜索关键词
- **用途**：在多个文件中搜索关键词，支持正则表达式
- **参数**：`path` (目录路径), `pattern` (搜索模式), `file_pattern` (文件类型过滤，如 "*.ts"), `recursive` (是否递归)
- **返回**：匹配结果列表（文件路径 + 行号 + 匹配内容）

### 4. `delete_file` - 删除文件或目录
- **用途**：删除单个文件或整个目录
- **参数**：`path` (文件/目录路径), `recursive` (删除目录时是否递归)
- **返回**：操作结果

### 5. `rename_file` - 重命名/移动文件
- **用途**：重命名文件或移动到新位置
- **参数**：`old_path` (原路径), `new_path` (新路径)
- **返回**：操作结果

## 优先级 2：文件内容操作

### 6. `write_file` - 写入文件内容
- **用途**：创建新文件或覆盖现有文件
- **参数**：`path` (文件路径), `content` (文件内容), `encoding` (可选)
- **返回**：操作结果

### 7. `append_to_file` - 追加内容到文件
- **用途**：在文件末尾追加内容
- **参数**：`path` (文件路径), `content` (要追加的内容)
- **返回**：操作结果

### 8. `replace_in_file` - 替换文件中的文本
- **用途**：在文件中查找并替换文本
- **参数**：`path` (文件路径), `search` (要查找的文本), `replace` (替换为的文本), `regex` (是否使用正则)
- **返回**：替换次数和结果

## 优先级 3：系统操作

### 9. `execute_command` - 执行系统命令
- **用途**：执行 PowerShell (Windows) 或 Shell (macOS/Linux) 命令
- **参数**：`command` (命令字符串), `working_dir` (工作目录，可选)
- **返回**：命令输出和退出码
- **⚠️ 安全**：需要沙箱机制，记录所有执行的命令

### 10. `get_system_info` - 获取系统信息
- **用途**：获取磁盘空间、内存使用等系统信息
- **参数**：无
- **返回**：系统信息对象

### 11. `get_disk_space` - 获取磁盘空间
- **用途**：检查各磁盘的可用空间
- **参数**：`drive` (可选，指定盘符如 "C:")
- **返回**：磁盘空间信息

## 优先级 4：高级功能

### 12. `batch_rename` - 批量重命名
- **用途**：根据模式批量重命名文件
- **参数**：`directory` (目录), `pattern` (匹配模式), `rename_template` (重命名模板)
- **返回**：重命名结果列表

### 13. `find_duplicates` - 查找重复文件
- **用途**：根据文件内容哈希查找重复文件
- **参数**：`directory` (目录), `recursive` (是否递归)
- **返回**：重复文件组列表

### 14. `clean_temp_files` - 清理临时文件
- **用途**：清理系统临时文件和缓存
- **参数**：`paths` (要清理的路径列表，可选)
- **返回**：清理结果
